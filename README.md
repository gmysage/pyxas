# pyxas 
pyxas is a python library for 2D XANES fitting

## Prerequisites
python 3.6\
numpy\
scipy\
sklean\
skimage\
tomopy\
pystackreg

## Installation
Clone this project to your machine:

```
git clone https://github.com/gmysage/pyxas.git
cd pyxas
python xanes_2D.py
```
