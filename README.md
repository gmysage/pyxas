# pyxas 
PyXAS is a python library for fast 2D/3D XANES analysis with graphic user interface (GUI) provided.


## Installation

Recommended:

```
pip install pyxas

```
And run "pyxas_gui"


Or clone this project to your machine::

```
git clone https://github.com/gmysage/pyxas.git
cd pyxas
./pyxas_gui
```
